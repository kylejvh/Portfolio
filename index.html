My Projects

MovieCards
React w/ Hooks Axios React Router Styled Components

React app that accesses The Movie Database (TMDb) API to display movie data.

This was my first major React project that I continuously worked on. I love great movies and I love great stories. This interest helped me stay persistently focused as I worked through various challenges developing the app. It was the culmination of many concepts I've learned in courses, books, and tutorials, and it was an extremely valuable learning tool. Another motivating factor driving this project was to a desire work with an API that I found interesting. TMDb's API provides access to a treasure trove of data, and allowed me the freedom to build out a feature-rich application.

  • • Uses React w/ Hooks: useEffect, useContext, useReducer, as well as a custom Axios hook for data fetching from the API.
  • • Uses Reach Router for routing.
  • • Styled with Styled Components

MaterialChat
React w/ Hooks socket.io+Node.js Material-UI

Real-time fully featured React chat application built with Material-UI and socket.io.

The idea of building this React app intrigued me because I am a big fan of Discord and use it frequently. My goal was to replicate some aspects of Discord's functionality and design, while keeping the scope of my initial project realistic. Using Material-UI allowed me to focus most of my efforts on implementing features, like username and chatroom name validation and typing/online user event notification.

  • • Uses React w/ Hooks: useEffect, useContext, useReducer, etc.
  • • Uses socket.io+Node.js with Express for client-server communication.
  • • Built with Material-UI components and theme system.

Utah Travel
HTML CSS JS lightgallery

A showcase website built around my trip to Utah in 2019.

This project started as a way to teach myself CSS Grid and Flexbox, with the goal of making responsive image galleries with each. Over time, I have used it as a playground project to try experimenting with CSS tricks and techniques I've learned. During my trip down the CSS rabbit hole, I implemented transitions, keyframe animations, background/text blend modes, and style changes via JavaScript. I took a lot of interesting pictures on this trip and wanted an elegant solution to show some of them off, so I also used lightgallery.js on a dedicated page.

  • • Built with HTML/CSS/JS - designed to be fully responsive.
  • • Heavy use of CSS - Grid, Flexbox, tranistions, animations, blend modes, etc.
  • • Uses lightgallery.js with image srcsets for fast, lightweight browsing of photos at all screen sizes
