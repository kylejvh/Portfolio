React / Redux Node.js / Express MongoDB / Mongoose REST API Socket.io Material-UI

MERN stack-based modern chat application built with Material-UI.

The idea of building a chat app intrigued me because I'm a big fan of Discord and use it frequently. My initial goal was to replicate some aspects of Discord's functionality and design, learning as much as possible along the way. Starting as a simple React app, I refactored it over time as I learned how to implement key back end features with a great looking front end interface.

  • • RESTful API with full CRUD operation support
  • • MongoDB for persistent user accounts, chatrooms, and messages
  • • Advanced Mongoose Models with Validation and Virtual Population
  • • JSON Web Token based authentication and protected routes
  • • Socket.io integration for real-time communication: typing notifications, active user lists, and instant messaging
  • • Uses React and Redux for front end interface and state management.
  • • Built with Material-UI components and theme system.

MovieCards
Live Demo Github
React / Redux Axios / TMDb API React Router Styled Components

React app that accesses The Movie Database (TMDb) API to display movie data.

This movie app has been an invaluable platform for learning everything related to front end development. I love great movies and I love great stories. This interest has helped to stay persistently focused while working through various development challenges. I've continuously implemented many concepts I've learned from courses, books, and tutorials. Another motivating factor was a desire to work with an API that I found interesting. TMDb's API provides access to a treasure trove of data, and allowed me the freedom to build out a feature-rich application.

  • • Uses React and Redux for front end interface and state management
  • • Uses Redux Thunk and Axios for asynchronous data fetching from TMDb API
  • • Uses React Router for routing
  • • Uses LocalStorage API for persistent favorites list of movies
  • • Heavy use of CSS-in-JS with Styled Components

Utah Travel
Live Demo Github
HTML CSS JS lightgallery

A showcase website built around my trip to Utah in 2019.

This is an earlier project aimed at developing skills with CSS Grid and Flexbox, with the goal of making responsive image galleries with each. Over time, I have used it as a playground project for experimenting with CSS tricks and techniques I've learned. During my trip down the CSS rabbit hole, I implemented transitions, keyframe animations, background/text blend modes, and style changes via JavaScript. I took a lot of interesting pictures on this trip and wanted an elegant solution to show some of them off, so I also used lightgallery.js on a dedicated page.

  • • Built with HTML/CSS/JS - designed to be fully responsive
  • • Heavy use of CSS - Grid, Flexbox, tranistions, animations, blend modes, etc
  • • Uses lightgallery.js with image srcsets for fast, lightweight browsing of photos at all screen sizes
